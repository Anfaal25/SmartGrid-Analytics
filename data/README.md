# Dataset Description – UCI Household Power Consumption

Original Source:
https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption

## Files

### 1. household_power_consumption.txt
Raw dataset with minute-level power measurements (2006–2010).

### 2. household_power_consumption.csv
CSV version generated by: **convert_txt_to_csv.py**

### 3. processed_power.csv
Preprocessed dataset:
- Generated by: **preprocess.py**
- Parsed datetime
- Numeric conversion
- Dropped missing values
- Resampled to hourly averages
